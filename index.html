JSSophia #5: Jeudi 21 Février 2013

  • Atelier HTML5 Games

    Corinne Krych Mathieu Bruyen Fabrice Matrat

    What were you dreaming to do as a kid? If you went into programming because of games, this session is tailored for you! Join the fun and discover how to build multi-players mobile games. We show you a template and you’ll do the coding. The example is based on KissingTurtles games (Grails48 hackathon project, see http://kissingturtles.cloudfoundry.com/). We will focus on 3 points:

    1. HTML5 canvas programming: canvas 2D API, browser animation effect
    2. Mobile programming with HTML5 and jQuery mobile, Angular etc..
    3. We then delve into multi-players challenges (using event-push Grails plugin).
